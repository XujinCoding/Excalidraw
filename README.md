# 如何使用本仓库并使用仓库画画呢
1. 下载VsCode

    https://code.visualstudio.com/download

2. 安装插件(本仓库使用的是支持中文手写体的Excalidraw-cj)

    ![Alt text](image.png)

3. clone 项目
    ```
    git clone https://github.com/XujinCoding/Excalidraw.git
    ```

    打开文件就可以尽情画图了
